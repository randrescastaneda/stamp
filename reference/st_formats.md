# Inspect available formats

Return a sorted character vector with the names of formats currently
registered in the internal registry.

## Usage

``` r
st_formats()
```

## Value

Character vector of format names.

## Examples

``` r
st_formats()
#> [1] "csv"     "fst"     "json"    "parquet" "qs"      "qs2"     "rds"    
```

# Project root directory recorded by st_init() (internal)

Return the absolute project root directory previously recorded by
[`st_init()`](https://randrescastaneda.github.io/stamp/reference/st_init.md).
If not set, defaults to the current working directory.

## Usage

``` r
.st_root_dir(alias = NULL)
```

## Value

Character scalar absolute path to project root.
